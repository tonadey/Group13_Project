236 /* Auto-sync is wired into every GUI change that affects what should be
237 * visible in VR (colour pick, visibility toggle, slider release,
238 * explode end, opacity reset). Each call resets a 200ms debounce;
239 * once the user pauses for that long, doVRSync() runs once. Without
240 * the debounce, dragging through 50 slider values would rebuild the
241 * VR scene 50 times in a second and tank headset framerate.
242 *
243 * doVRSync() is the immediate path - it's what the manual Sync button
