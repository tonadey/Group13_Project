Detailed Description

Main application window for the CAD and VR viewer.

MainWindow connects the Qt interface to the VTK rendering scene. It handles file loading, part selection, colour and visibility changes, clip/shrink filters, exploded view controls, opacity controls, screenshots, and VR start/stop/sync actions.

◆ updateRender()

void MainWindow::updateRender ( )
private

Updates the desktop render window after changes to the model tree.

Diffs the tree actors against the renderer and updates the scene.

The renderer is updated by comparing the current scene actors with the actors required by the tree, so only added or removed actors are changed.
