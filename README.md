## Built With

This project is developed using the following core technologies:

- **OpenVR** — Virtual reality hardware integration  
- **CMake 3.x** — Build configuration and project generation  
- **C++** — Primary programming language

## Usage

Once the application is installed and running, you can use it to load, view, and modify CAD models.  
Below are the main actions available in the software:

- Import STL files using the toolbar buttons  
- Organise and manage model parts through the TreeView  
- Right‑click items to rename, recolour, or toggle visibility  
- Apply visual filters such as **Clip** and **Shrink**  
- Adjust lighting and other display settings  
- Enter VR mode using **Start VR**  
- Exit VR mode using **Stop VR**  

These tools allow you to explore CAD models both on the desktop and in an immersive VR environment.
